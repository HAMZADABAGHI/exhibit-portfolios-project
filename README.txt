Exhibit is an academic project developed with a colleague, the purpose is to render a service of the creation of your own portfolio and make it available to other users. 

The project follows the three-layer architecture and developed with the technologies below :

Spring Boot | Spring MVC | Spring Data | JPA | Hibernate | MySQL | Spring Security | Bootstrap 4
